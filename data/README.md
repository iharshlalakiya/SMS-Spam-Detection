# Dataset Information

## SMS Spam Collection Dataset

### Overview
This directory contains the SMS Spam Collection dataset used for training and evaluating spam detection models.

### Dataset Details

| Attribute | Value |
|-----------|-------|
| **Name** | SMS Spam Collection |
| **Source** | UCI Machine Learning Repository |
| **Size** | 5,574 SMS messages |
| **Format** | CSV |
| **Classes** | 2 (ham, spam) |

### Class Distribution
- **Ham (legitimate)**: 4,827 messages (86.6%)
- **Spam**: 747 messages (13.4%)

### Download Instructions

1. Download from Kaggle:
   - URL: https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset
   - Download and extract spam.csv
